# Лендинг для продажи чат-ботов и ИИ-агентов

Современный лендинг для продажи услуг по созданию чат-ботов, ИИ-агентов и интеграций на n8n.

## 🚀 Особенности

- **Современный дизайн** с градиентными цветами и AI-фонами
- **Адаптивная верстка** для всех устройств
- **Интерактивные элементы** с анимациями Framer Motion
- **Форма заявки** с динамическими полями и валидацией
- **Всплывающий чат** с AI-ассистентом
- **Интеграция с n8n** для автоматизации процессов
- **Google Analytics 4** для отслеживания конверсий
- **Honeypot защита** от спама
- **SEO оптимизация**

## 📋 Структура лендинга

1. **Hero-блок** - главный экран с призывом к действию
2. **Преимущества** - карточки с ключевыми выгодами
3. **Вкладки тарифов** - 3 типа услуг с описанием
4. **Как мы работаем** - 4 этапа работы
5. **Кейсы** - примеры успешных проектов
6. **Отзывы** - отзывы клиентов
7. **Тарифы** - детальная таблица цен
8. **FAQ** - ответы на частые вопросы
9. **Форма заявки** - сбор лидов
10. **Футер** - контакты и ссылки

## 🛠 Технологии

- **React 18** - основной фреймворк
- **Vite** - сборщик проекта
- **Tailwind CSS** - стилизация
- **Framer Motion** - анимации
- **React Hook Form** - работа с формами
- **React Hot Toast** - уведомления

## 🚀 Запуск проекта

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Соберите для продакшена:
```bash
npm run build
```

## ⚙️ Настройка

### n8n Webhook

Замените URL webhook в файлах:
- `src/utils/n8nIntegration.js`
- `src/components/ContactForm.jsx`
- `src/components/ChatPopup.jsx`

### Google Analytics

Замените `GA_MEASUREMENT_ID` в `index.html` на ваш ID.

### Контакты

Обновите контактную информацию в `src/components/Footer.jsx`.

## 📊 KPI и метрики

- Конверсия в заявку ≥ 5%
- Дочитываемость до блока «Готовы начать?» ≥ 60%
- Отслеживание событий GA4:
  - `form_submit` - отправка формы
  - `chat_opened` - открытие чата
  - `tab_switch` - переключение вкладок
  - `scroll_depth` - глубина скролла
